
# ReservaPro - User Flows

## 1. Client Flow (Discovery & Booking)
1. **Discovery:** User visits Landing Page, clicks "Explorar Directorio".
2. **Search:** Filter professionals by specialty (e.g., Psychology) or name.
3. **Profile:** Click "Ver Perfil" to see the public professional page.
4. **Booking:** 
   - Select a Service.
   - Select a Date and Time Slot.
   - Click "Confirmar Reserva" (prompts login if not authenticated).
5. **Management:** Client views "Mis Reservas" in their dashboard.

## 2. Professional Flow (Business Management)
1. **Catalog:** Go to "Mis Servicios", create/edit/delete services.
2. **Agenda:** Go to "Agenda Maestra" to see upcoming appointments.
3. **Control:** Approve "Pending" requests or mark "Confirmed" ones as "Completed" or "No-Show".
4. **CRM:** Go to "Mis Clientes" to see registered users or add external clients manually.
5. **Insights:** View AI predictions for no-shows and business KPIs in the Main Dashboard.

## 3. SuperAdmin Flow (Platform Oversight)
1. **Command Center:** Monitor global metrics (Total MRR, Active Tenants).
2. **Compliance:** Hide/Show professionals in the public directory based on content quality.
3. **Billing Simulator:** Expire or Activate subscriptions manually to test system response.
